# odinfw

## Usage
### Create a war file
```
gradlew clean bootWar
```
### Create an executable war file
```
gradlew clean bootWar
```
### Create an executable jar file
```
gradlew clean bootJar
```
### Start the demo app
After the app started, visit http://localhost:8080/odinfw
```
gradlew clean bootRun
```
## FAQ
